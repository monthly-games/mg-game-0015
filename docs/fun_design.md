## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 폐허 탐사/자원 수집                                  │
│         ↓                                               │
│  2. 건물/시설 건설 및 업그레이드                         │
│         ↓                                               │
│  3. 주민 관리/스토리 진행                                │
│         ↓                                               │
│  4. 왕국 확장/정책 결정 ─────────────────→ (1로 반복)     │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **폐허 탐사**: 방치/탐험으로 건설 자원 수집
2. **건설/업그레이드**: 건물 복원, 시설 레벨업
3. **주민 관리**: 주민 배치, 스토리 퀘스트 진행
4. **왕국 확장**: 새 지역 해금, 정책/방향 결정

## 3. 재미 축 (Fun Pillars)

### 3.1 장기 성장 (Long-term Growth)
- **목표**: 오랜 시간에 걸친 대규모 성취감
- **핵심 경험**: 폐허가 번영하는 왕국이 되는 대서사
- **주요 요소**:
  - 단계별 왕국 성장 (폐허→마을→도시→왕국)
  - 장기 목표 시스템
  - 마일스톤 보상
  - 성장 기록/앨범

### 3.2 도시/왕국 꾸미기 (Expression)
- **목표**: 나만의 왕국을 설계하는 재미
- **핵심 경험**: 내가 만든 왕국에 대한 자부심
- **주요 요소**:
  - 건물 배치 자유도
  - 테마 건축물
  - 랜드마크 시스템
  - 왕국 외관 커스터마이징

### 3.3 서사 (Narrative)
- **목표**: 왕국 재건의 감동적인 스토리 경험
- **핵심 경험**: 왕국과 주민들의 이야기에 몰입
- **주요 요소**:
  - 메인 스토리라인
  - 주민 개인 스토리
  - 선택형 이벤트
  - 시즌 스토리

## 4. 디자인 레버 (Design Levers)

### 4.1 단계별 성장
| 레버 | 설명 |
|------|------|
| city_stage 구분 | 폐허→번영으로 이어지는 city_stage 단계를 명확히 구분 |
| 구현 방향 | 5단계: 폐허→정착지→마을→도시→왕국, 단계별 비주얼 완전 변경 |

### 4.2 스토리/이벤트 연동
| 레버 | 설명 |
|------|------|
| 건물-스토리 연결 | 건물/시설 업그레이드에 스토리/이벤트 연동 |
| 구현 방향 | 대장간 완성 → 대장장이 NPC 합류, 도서관 건설 → 역사 퀘스트 해금 |

### 4.3 선택형 정책
| 레버 | 설명 |
|------|------|
| 왕국 방향성 결정 | 선택형 스토리/정책으로 왕국 방향성 결정(성장 vs 복지 등) |
| 구현 방향 | 군사 투자 → 방어력 UP/행복도 DOWN, 복지 투자 → 인구 UP/자원 DOWN |

## 5. 재미 측정 지표 (KPI)
