- **수익화 우선순위**: IAP = 광고 (균형)

## 1. 광고 전략

### 1.1 보상형 광고 배치

| Placement | 상황 | 보상 | 일일 제한 |
|-----------|------|------|----------|
| `build_speed` | 건설 대기 | 시간 50% 단축 | 10회 |
| `resource_double` | 자원 수집 | 2배 | 5회 |
| `quest_reward` | 퀘스트 완료 | 보상 2배 | 5회 |
| `daily_bonus` | 일일 보너스 | 2배 | 1회 |
| `story_unlock` | 스토리 대기 | 즉시 해금 | 3회 |

### 1.2 전면 광고

- 세션 복귀 시 (30분 이상)
- 쿨다운 5분
- 구독자 제외
