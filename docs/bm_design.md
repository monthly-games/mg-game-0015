## 3. 상세 BM 설계

### 3.1 건설 가속/확장 BM

#### 무료 구조
- 방치 자원으로 건설 진행
- 무료 속성 1회/일
- 레벨업으로 새 건물 해금

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 즉시 완료 | 건설 시간 스킵 | 프리미엄 재화 |
| 건설 큐 추가 | 동시 건설 +1 | $2.99 |
| 자원 패키지 | 건설 자원 대량 | $4.99~$9.99 |
| 왕국 확장권 | 새 지역 즉시 해금 | $4.99~$9.99 |

### 3.2 꾸미기/테마 BM

#### 무료 구조
- 스토리 진행으로 기본 건물 외형 획득
- 업적 달성으로 특수 장식 해금
- 이벤트에서 한정 테마 파밍

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 테마 패키지 | 왕국 테마 세트 | $6.99~$14.99 |
| 랜드마크 | 특수 건물/조형물 | $2.99~$4.99 |
| 시즌 테마 | 계절/시대별 테마 | $9.99 |
| 왕국 배경 | 전체 배경 변경 | $3.99 |

### 3.3 캐릭터/영웅 BM

#### 무료 구조
- 스토리 진행으로 기본 영웅 5종 합류
- 일일 무료 뽑기 1회
- 퀘스트로 영웅 조각 파밍

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 영웅 뽑기 | 10연차 1회 | 프리미엄 재화 2,500 |
| 천장 시스템 | 70연차 SSR 확정 | - |
| 선택권 패키지 | 10연차 + SR 선택권 | $9.99 |
| 영웅 스킨 | 외형 변경 | $3.99~$6.99 |

### 3.4 방치 수익 BM

#### 무료 구조
- 오프라인 자원 수집 상한: 12시간
- 왕국 수익 누적

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 수익 2배 | 광고 시청 시 적용 | 무료 (광고) |
| 수익 3배 패스 | 상시 적용 (30일) | $4.99/월 |
| 오프라인 확장 | 24시간까지 누적 | 패스 포함 |

### 3.5 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 자원 2배, 건설 단축, 무료 뽑기 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $2.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 콘텐츠 무과금으로 도달 가능 (시간 차이만)
- 영웅 성능보다 왕국 발전도가 중요
- 경쟁 요소 최소화 (개인 성장 중심)

### 과금 압박 최소화
- 장기 플레이 설계로 급한 과금 불필요
- 일일 플레이만으로 꾸준한 성장
- 광고로 핵심 자원 충분히 획득
