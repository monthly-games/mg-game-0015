## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 건설/방치 시스템 밸런스 |
| 7~10주 | 스토리 1부 완성 |
| 11~12주 | 첫 왕국 위기 이벤트 준비 |

**핵심 과제:**
- 건설 성장 곡선 최적화
- 스토리 몰입도 강화
- 방치 수익 밸런스

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 왕국 위기 이벤트 1차 (침공) |
| 5개월 | 스토리 2부 시작 |
| 6개월 | 왕국 위기 이벤트 2차 (재난) |
| 7개월 | 신규 건물/지역 추가 |
| 8개월 | 스토리 2부 완결 + 여름 이벤트 |

**핵심 과제:**
- 시즌별 왕국 위기 이벤트
- 스토리와 건설 콘텐츠 동기화
- 지역 확장

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 0019/0020/0024 세계관 연계 |
| 10개월 | 스토리 3부 시작 |
| 11개월 | 대규모 왕국 위기 |
| 12개월 | 1주년 이벤트 + 스토리 3부 완결 |

**핵심 과제:**
- 크로스 게임 세계관 연계
- 스토리 확장
- 장기 유저 엔드콘텐츠
