## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 맵 이동 | 맵 영역 | 드래그 |
| 줌 인/아웃 | 맵 영역 | 핀치 |
| 건물 선택 | 건물 위 | 탭 |
| 건물 배치 | 빈 영역 | 탭 (건설 선택 후) |
| 건물 이동 | 건물 위 | 롱프레스 → 드래그 |

### 터치 피드백
- 건물 선택: 정보 팝업
- 배치 가능: 녹색 그리드
- 배치 불가: 빨간 그리드
- 건설 완료: 파티클 + 팡파레
